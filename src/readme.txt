
Code corresponding to the paper: 

M. K. Titsias, P. Dellaportas.
Gradient-based Adaptive Markov Chain Monte Carlo. In Proceedings of the 33th Conference on Neural Information Processing Systems (NeurIPS), 2019.

The main routines for the proposed algorithms are in the directory mcmc:
gad_mala_exact.m : gadMALAe
gad_mala_fast.m  : gadMALAf
gad_rwm.m  : gadRWM

THe demo files run all the experiments in the paper. See individual files for 
specific details.
